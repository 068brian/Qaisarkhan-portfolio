EMBEDDED CITIATION

EMBEDDED CITATION MAPPING AND DATA ENHANCEMENT FOR AUSTLII

Project Type: Application Development Projects

Last Updated On: May 05, 2017

Categories: Application Development Projects

Tags: Development, PERL, HTML, CSS, JQuery, AJAX

Author: Qaisar Khan

Client: Federal Court of Australia (AustLII)

Project Overview

The Australasian Legal Information Institute (AustLII) provides free internet access to Australasian legal materials. AustLII’s mission is to improve access to justice through better information availability. It is one of the largest legal content repositories on the web with over 4 million searchable documents and more than 700,000 daily hits.

Many legal records from 1977 to 1996 used embedded citations, but most of these had missing or incomplete references. Around 12 years of legal documents lacked references entirely, and the remaining 8 years had only partial citations. The goal was to develop a system to accurately find and map complete citations across 20 years of legal documents.

Developed Solution

All documents were provided as HTML files. A reference database dump with known citations (provided by AustLII) was used as a baseline, although it too had inconsistencies due to human data entry errors.

A complex citation matching algorithm was created using Object-Oriented Perl and Shell Scripting. This algorithm extracted key citation elements from each HTML file and compared them with the reference database using fuzzy matching techniques. Pattern-matching and context verification were used to identify and confirm accurate citations.

The final solution automated the identification, verification, and correction of citations across all documents. It significantly improved accuracy and eliminated the manual effort previously required. Each corrected citation was registered in an output Registry File, giving users access to proper legal references.
